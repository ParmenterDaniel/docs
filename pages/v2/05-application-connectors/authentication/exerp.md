## Cyclr setup

### Account setup

You will be asked for the following values when installing the Exerp connector within an account:

| Value        | Description                              |
| :----------- | :--------------------------------------- |
| **Username** | The username of your Exerp account.      |
| **Password** | The client secret of your Exerp account. |
